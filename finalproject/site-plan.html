Overview

Purpose

The purpose of this site is to be a crash course on the fundamentals of mathematics.

Audience

The target audience for this site includes anyone who has a hard time with math concepts and could use examples and analogies to help them grasp it.

Dynamic elements

The addition/subtraction section will feature a drag-&-drop exercise including a number line and blocks that represent different positive and negative values.

Wireframes

Create two wireframes for your site. One for each page and list them here

Home

The FAQs will be in the usual style for FAQs on a website: An array of expandable boxes with just the question in the title and the answer in the collapsible section.

Addition/subtraction page

Each of the four panels will expand when tapped/clicked on and fill most of the screen.
The first activity has numbers represented by rectangular blocks which can be dragged and droped from the yellow box onto the number line.
-- Red rectangles are positive numbers, blue ones are negative numbers.

The second activity is similar, but the user will construct each number block using the slider at the bottom.
